# Changelog

Alle wesentlichen Änderungen dieses Projekts werden hier dokumentiert.

## [Unreleased]

- GitHub-fertige Repository-Struktur ergänzt
- Automatische PlatformIO-Builds für ESP32 und ESP32-S3 ergänzt
- Automatischer Release-Build für Versions-Tags ergänzt
- Dokumentation, Issue-Vorlagen und Sicherheitsrichtlinien ergänzt

## [0.1.2]

- Notfall-Access-Point startet erst nach drei fehlgeschlagenen WLAN-Verbindungsversuchen
- Access Point wird nach erfolgreicher WLAN-Verbindung automatisch abgeschaltet
- DHCP und manuelle IPv4-Konfiguration ergänzt
- Aktuelle DHCP-Werte können in die statischen Eingabefelder übernommen werden

## [0.1.1]

- WLAN-Suche mit SSID-Auswahl ergänzt
- Klassischen ESP32 als Standard-PlatformIO-Umgebung eingerichtet

## [0.1.0]

- Erste Version mit Weboberfläche, BLE-Suche, mehreren Speichern, MQTT und Home Assistant Discovery
